## WorldWise — Track Your Travels Around the World 🌍

A travel tracking web application that lets you log cities you’ve visited, visualize them on an interactive map, and keep a personal record of your journeys. Built with React and modern state management, focusing on maps, geolocation, and clean UI architecture.

## Features

🌍 Interactive map with your visited cities  
📍 Add cities by selecting a location on the map  
🗺️ View all visited places in a structured list  
✏️ Add notes and details for each city  
🧭 Geolocation support to detect your position  
💾 Persistent data storage (saved visits)  
🧠 Clean state management with React hooks and Context API  
⚡ Fast and responsive UI experience

---

## How to Use

1. Open the app and allow location access (optional)
2. Click on the map to select a city
3. Add details about your visit
4. Save the city to your travel list
5. View and manage all visited locations
6. Explore your journey visually on the map

---

## Tech Stack

- React (functional components, hooks: useState, useEffect, useContext, useReducer)
- React Router
- Leaflet / Map library for interactive maps
- JavaScript (ES6+)
- CSS3
- Local storage / fake API for persistence

---

## Running Locally

```bash
npm install
npm run dev   # start development server
npm run build # production build
```
